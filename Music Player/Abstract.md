# Abstract

## Music Player — Desktop Application Using Python

This project implements a desktop Music Player application using Python, built with
customtkinter for the graphical user interface, python-vlc as the audio playback engine,
and mutagen for reading MP3 metadata including embedded album art. The application allows
users to load a folder of MP3 files, navigate between tracks, play and pause music, view
album artwork extracted from ID3 tags, and automatically advance to the next song upon
completion. The interface follows a modern, minimalist design aesthetic featuring a cream
and purple color scheme, rounded card layout, and intuitive playback controls. The entire
application is encapsulated in a single Python file of approximately 100 lines, demonstrating
efficient use of third-party libraries to build a polished, lightweight desktop tool.

**ABSTRACT**

This project presents a desktop music player application developed in Python, leveraging libraries beyond the standard syllabus. The player offers a modern interface, playlist management, lyrics display, and advanced playback controls. It demonstrates practical use of Python for multimedia applications, integrating third-party libraries for audio processing, metadata extraction, and GUI enhancements. The solution is stable, user-friendly, and extensible for future features.
